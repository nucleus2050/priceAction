## 🏗️ 技术架构

### 技术栈
```
前端层: 命令行界面 (CLI)
    ↓
应用层: Python API
    ↓
核心层: 识别引擎
    ├── OpenCV (图像处理)
    ├── PaddleOCR (文字识别)
    └── NumPy (数值计算)
    ↓
数据层: 多格式输出
    ├── JSON
    ├── CSV
    └── Excel
```
